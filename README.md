<Purpose>
The purpose of the program is to address the growing challenge where hospitals struggle to maintain adequate records as well as to properly track the status of doctors' apointments
It was created with the aim of completly virtualizing the hospial system- offering capabilities such as doctor management, patient manual as well as the appointmentmanagement on both the hospital and doctor level.
